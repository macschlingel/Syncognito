Powerful Features, Minimal Footprint

Real-time Monitoring

Automatically detects file changes and syncs them instantly using FSEvents. No more manual copying.

Flexible Configuration

Set up multiple sync jobs with custom intervals. Choose between real-time sync or scheduled updates.

Privacy First

Runs entirely on your Mac. No cloud services, no accounts, no data collection. Your files stay yours.

Menubar Integration

Lives in your menubar, always accessible. See sync status at a glance with animated indicators.

Progress Tracking

See real-time progress with file counts and current operation. Know exactly what's being synced.

Multiple Jobs

Sync multiple folder pairs simultaneously. Perfect for backing up to multiple destinations.

How It Works

1

Add Sync Jobs

2

3

Set & Forget

Syncognito runs in the background, keeping your folders synced automatically.
